# Sanity (sanity)

Installs a proper set of common command line utilities for, you know, keeping your sanity working with devcontainers.

## Example Usage

```json
"features": {
    "ghcr.io/janw/devcontainer-features/sanity:0": {}
}
```

## Options

No options, YGWYG.

## OS Support
